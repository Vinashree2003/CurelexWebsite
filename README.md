# Curelex - HealthTech Solutions

A modern, responsive website for Curelex, a leading healthtech company dedicated to transforming healthcare through innovative technology solutions.

## Technologies Used

- **HTML5**: Semantic markup for structure and content
- **CSS3**: Styling with responsive design, animations, and modern layouts
- **JavaScript**: Interactive features including team member display and navigation
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Inter font family for typography

## Features Implemented

- **Responsive Navigation**: Hamburger menu for mobile devices with smooth transitions
- **Hero Section**: Eye-catching landing area with call-to-action buttons and hero image
- **About Section**: Company overview with key statistics (patients served, partners, satisfaction rate)
- **Services Grid**: Showcase of core services including Telemedicine, Health Monitoring, AI Diagnostics, and Genetic Analysis
- **Interactive Team Section**: Dynamic display of team members with bubble animation and role-based navigation
- **Contact Form**: User-friendly form for inquiries with contact information
- **Footer**: Social media links, quick navigation, and company branding
- **Mobile-First Design**: Optimized for all device sizes with CSS Grid and Flexbox

## How to View

Simply open `index.html` in any modern web browser to view the website.
